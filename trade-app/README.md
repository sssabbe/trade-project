# База данных "Цветочный магазин"

## Описание проекта
Проект представляет собой базу данных для управления цветочным магазином.
Система позволяет вести учет товаров, управлять клиентской базой, обрабатывать заказы и отслеживать продажи.
База данных разработана для автоматизации основных бизнес-процессов цветочного магазина.

## Основные функции
- Учет цветов и сопутствующих товаров
- Управление клиентской базой
- Оформление и отслеживание заказов
- Учет продаж и остатков товаров
